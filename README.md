# Projeto CRUD Aluno

Sistema de gerenciamento de alunos com Express + Frontend.

## Como executar

1. Instalar dependências:

```bash
cd backend
npm install
```

2. Executar servidor:

```bash
npm run dev
```

3. Acessar: http://localhost:3000

## Funcionalidades

- Cadastrar aluno
- Listar alunos
- Editar aluno
- Excluir aluno
- Validações
- Layout responsivo
